# Java Swing To-Do App

## 📌 Project Overview

The **Java Swing To-Do App** is a simple desktop application developed using **Java Swing**. It allows users to manage daily tasks through an easy-to-use graphical user interface (GUI). Users can add new tasks and delete selected tasks from the list.

## 🎯 Objective

To build a basic desktop To-Do List application using Java Swing and understand GUI development, event handling, and layout management in Java.

## 🛠 Technologies Used

* Java
* Java Swing
* AWT (Layout Managers)
* VS Code / IntelliJ IDEA CE / Eclipse

## ✨ Features

* Add new tasks
* Delete selected tasks
* Scrollable task list
* Simple and user-friendly GUI
* Input validation for empty tasks

## 📂 Project Structure

```
ToDoApp/
│
└── ToDoApp.java
```

## ▶ How to Run


### Using IntelliJ IDEA CE

1. Create a new Java project.
2. Add `ToDoApp.java`.
3. Run the application.



## 🖥 GUI Components Used

* `JFrame` – Main application window
* `JPanel` – Organizes components
* `JTextField` – Task input field
* `JButton` – Add and Delete buttons
* `JList` – Displays tasks
* `DefaultListModel` – Stores tasks
* `JScrollPane` – Provides scrolling for the task list
* `JOptionPane` – Displays messages to the user

## 📚 Concepts Covered

* Java Swing
* Event Handling (`ActionListener`)
* Layout Managers (`BorderLayout`)
* GUI Components
* Object-Oriented Programming (OOP)
* Event Dispatch Thread (EDT)
